# Лабораторна робота №6

## Назва дисципліни
Інструменти та засоби програмування

## Опис репозиторію
Даний репозиторій містить виконані лабораторні роботи з дисципліни «Інструменти та засоби програмування».  
У межах курсу розглядаються сучасні інструменти розробки програмного забезпечення, включаючи системи контролю версій Git, автоматизацію процесів розробки, тестування та документування проєктів.

## Структура репозиторію
Репозиторій організовано наступним чином:

- `lab1/` — перша лабораторна робота
- `lab2/` — друга лабораторна робота
- `lab3/` — третя лабораторна робота
- `lab4/` — четверта лабораторна робота
- `lab5/` — п’ята лабораторна робота
- `docs/` — документація проєкту

## Мета роботи
Метою даної лабораторної роботи є:
- ознайомлення з принципами роботи GitHub Pages;
- налаштування публікації документації;
- формування навичок ведення технічної документації у форматі Markdown;
- закріплення знань щодо роботи з системою контролю версій Git.

## Використані технології
- Git / GitHub
- Markdown
- GitHub Pages

## Результат роботи
У результаті виконання лабораторної роботи було:
- створено структуру документації;
- додано файл `index.md`;
- налаштовано публічний доступ до документації через GitHub Pages;
- отримано публічне посилання на документацію.
